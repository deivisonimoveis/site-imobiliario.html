Como não posso fazer isso remotamente (não tenho acesso às suas contas), você precisará seguir exatamente como descrito. Se ficar preso em algum passo, me diz qual e ajudo mais. ### Pré-requisitos - Certifique-se de que o domínio `deivisonlemosimoveis.com.br` está registrado no [Registro.br](https://registro.br). Se não estiver, registre lá primeiro (custa cerca de R$ 40 por ano). - Escolha uma hospedagem gratuita ou paga. Para este site simples (HTML/CSS/JS estático), recomendo algo gratuito como **GitHub Pages** (é ótimo para começar sem custos) ou **Vercel** (fácil e rápido). Se quiser algo mais profissional com backend futuro, use **Hostinger** (custa cerca de R$ 20/mês). Vou explicar duas opções: 1) Github Pages (gratuito e simples) e 2) Hospedagem paga para algo mais robusto. Escolhe a que preferir. --- ### **Opção 1: Usar GitHub Pages (Gratuito, Recomendado para Iniciantes)** Isso hospeda seu site diretamente no GitHub sem custos. Funciona bem para sites estáticos. #### Passo 1: Criar uma conta no GitHub - Acesse [github.com](https://github.com) e clique em "Sign up" (ou "Inscrever-se" em português). - Preencha seu nome, e-mail, e senha. Verifique seu e-mail (eles mandam um código). - Configure seu perfil básico (foto, bio opcional). #### Passo 2: Criar um repositório - No painel do GitHub, clique no ícone "+", selecione "New repository" (Novo repositório). - Dê o nome: `deivisonlemosimoveis.com.br` (ou qualquer nome, mas deixe em minúsculas sem espaços). - Marque "Public" (Público) e clique em "Create repository". - Anote a URL do repositório (ex: https://github.com/seu-usuario/deivisonlemosimoveis.com.br). #### Passo 3: Subir o arquivo do site - Use o botão "Code" -> "Download ZIP" do arquivo que criei acima (`index.html`) e baixe em seu computador. Abra o arquivo em um editor como Bloco de Notas ou VS Code. - Abra seu repositório no GitHub, clique "Add file" -> "Upload files". - Arraste o arquivo `index.html` para a área de upload. - Renomeie o arquivo no upload para `index.html` se não estiver. - Clique "Commit changes" (Confirmar mudanças) com uma mensagem como "Adicionando site inicial". #### Passo 4: Ativar GitHub Pages - No repositório, vá para "Settings" (Configurações). - Role para baixo até "Pages" na coluna esquerda. - Em "Source", selecione "Deploy from a branch". - Selecione branch "main" e pasta "/ (root)". - Clique "Save". - Aguarde 5-10 minutos. Eles vão gerar uma URL como `https://seu-usuario.github.io/deivisonlemosimoveis.com.br` (espelho dev do seu domínio real). #### Passo 5: Conectar ao seu domínio - No painel do GitHub, ainda na seção "Pages", clique "Add a domain" ou "Adicionar domínio personalizado". - Digite: `www.deivisonlemosimoveis.com.br` (inclua o www). - No Registro.br (painel do seu domínio): - Faça login em [registro.br](https://registro.br). - Vá para "Meus domínios" e selecione seu domínio. - Em "Configurações DNS", configure registros CNAME: - Nome: `www` - Tipo: CNAME - Dados: `seu-usuario.github.io` - Também adicione um registro A para a raiz (sem www): - Nome: vazio (ou @) - Tipo: A - Dados: `185.199.108.153` (IP do GitHub Pages; adicione 4 registros com IPs: 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153). - Salve e aguarde 24-48 horas para propagação DNS. #### Passo 6: Testar - Abra `www.deivisonlemosimoveis.com.br` no navegador. Deve carregar seu site. - Teste o formulário: Ele mostra uma mensagem de sucesso simples (mas não envia e-mail real; para isso, integra com Formspree ou EmailJS mais tarde). **Nota:** GitHub Pages tem limite de 100GB/mês, mas para seu site é mais que suficiente. --- ### **Opção 2: Hospedagem Paga (Hostinger - Mais Profissional)** Para controle total, use uma hospedagem paga. #### Passo 1: Contratar Hospedagem - Vá para [hostinger.com.br](https://hostinger.com.br) e clique "Hospedagem Compartilhada". - Escolha um plano básico (R$ 19.99/mês) e domínio (opÇÃO existente). - Cadastre-se com seu e-mail, cartão de crédito (ou boleto para teste). - Eles criarão seu painel (hPanel) automaticamente. #### Passo 2: Upload do Arquivo - Faça login no seu painel Hostinger. - Clique "Arquivos" (ou File Manager). - Na pasta `public_html`, clique "Upload" e carregue o arquivo `index.html`. - Renomeie para `index.html` se necessário (deve ser assim por padrão). #### Passo 3: Conectar o Domínio - Ainda no painel, vá para "Domínios" -> "Adicionar Domínio". - Digite `deivisonlemosimoveis.com.br`. - No Registro.br: - Acesse "Meus domínios" -> seu domínio. - Em "Configurações DNS", configure Nameservers: - NS1: ns1.hostinger.com (ou o fornecido por Hostinger). - NS2: ns2.hostinger.com. - Pode demorar até 24 horas. #### Passo 4: Configurar SSL (HTTPS) - No painel Hostinger, vá "Sites" -> "Gerenciar" seu domínio. - Clique "SSL" e ative SSL gratuito (Let's Encrypt). #### Passo 5: Testar - Acesse `www.deivisonlemosimoveis.com.br`. - Certifique-se de que o site carrega e é seguro (HTTPS). --- ### Dicas Gerais para Ambas as Opções - **Propagação DNS:** Pode levar 24 horas ou mais para o domínio apontar corretamente. Use ferramentas como [dnschecker.org](https://dnschecker.org) para verificar progresso. - **Formulário Funcional:** O site tem um formulário básico que mostra uma mensagem de sucesso. Para enviar e-mails reais: - Use [Formspree](https://formspree.io): Substitua `action="#"` no form por `https://formspree.io/f/YOUR_FORM_ID` e configure seu e-mail. - Ou integre EmailJS (biblioteca JS) para envio via serviço externo. - **Backups:** Sempre baixe o arquivo `index.html` do seu computador. - **Erro Comum:** Se a página não carregar, verifique se o arquivo está na pasta certa (raiz) e se o nome é exatamente `index.html`. - **Suporte:** Se usar GitHub/Vercel, leia documentação oficial: [pages.github.com](https://pages.github.com). Para Hostinger, use chat ao vivo no site deles. Se seguir estes passos e não funcionar, me diga exatamente onde parou (ex: "não consegui criar conta no GitHub") e ajudo com mais detalhes.
